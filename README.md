# 🎓 Sistema Acadêmico

Sistema de gerenciamento acadêmico desenvolvido em Java com interface gráfica e banco de dados MySQL, criado como projeto acadêmico.

---
 📖 Sobre

O sistema permite o gerenciamento de alunos, cursos, disciplinas, notas e faltas de uma instituição de ensino, oferecendo uma interface simples e intuitiva para o usuário.

 🛠️ Tecnologias

- **Java** Linguagem principal
- **MySQL** como banco de dados
- **JDBC** para comunicação com o banco
- **Eclipse IDE** como ambiente de desenvolvimento
